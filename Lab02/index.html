Curso: Programación de Internet

Descripción

Este laboratorio se centra en la creación de páginas web utilizando HTML y CSS.

El objetivo es comprender la estructura básica de una página y aplicar estilos con CSS.

Tecnologías Aprendidas

  • HTML
  • CSS
  • JavaScript

Objetivos

  1. Aprender etiquetas HTML básicas.
  2. Aplicar estilos CSS correctamente.
  3. Comprender la relación entre estructura y diseño web.
